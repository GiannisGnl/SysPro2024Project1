Αυτό το ReadMe αρχείο περιέχει πληροφορίες σχετικά με τον τρόπο λειτουργίας και
τη χρήση των Αρχείων: jobCommander, jobExecutorServer

jobCommander:

Εκτέλεση: Εκτελέστε το πρόγραμμα χρησιμοποιώντας την εντολή: ./jobCommander <εντολή> [παράμετροι].

Εντολές: Οι διαθέσιμες εντολές είναι οι εξής:

issueJob: Εκδίδει μια εργασία στον server.
setConcurrency: Ορίζει την ταυτόχρονη εκτέλεση εργασιών.
stop: Σταματά μια εκτελούμενη εργασία με βάση το ID της.
poll: Επιστρέφει την κατάσταση των εργασιών (running ή queued).
exit: Τερματίζει τον server και εξέρχεται από το πρόγραμμα.

Λειτουργίες:
establish_pipe: Δημιουργεί ένα named pipe για την επικοινωνία με τον server.
is_server_running: Ελέγχει αν ο server εκτελείται.
activate_server: Ενεργοποιεί τον server εάν δεν εκτελείται.
transmit_command: Μεταφέρει μια εντολή στον server μέσω named pipe.
Αυτές οι λειτουργίες επιτρέπουν την επικοινωνία μεταξύ του jobCommander και του jobExecutorServer για τη διαχείριση και εκτέλεση εργασιών.

jobExecutorServer:

Ο Job Executor Server είναι ένας εξυπηρετητής που διαχειρίζεται εργασίες και εκτελεί εντολές 
από μια ουρά εργασιών μέσω FIFO (First In First Out).

Εκτέλεση: Εκκίνηση του Server: Εκτελέστε τον server χρησιμοποιώντας την εντολή ./jobExecutorServer ή πολύ απλά
τρέχοντας τον jobCommander.

Λειτουργίες:

generate_pid_file: Δημιουργεί το αρχείο PID για τον server.
remove_pid_file: Αφαιρεί το αρχείο PID.
configure_fifo: Διαμορφώνει το FIFO για επικοινωνία.
add_job: Προσθέτει μια εργασία στην ουρά αναμονής.
start_next_job: Ξεκινά την επόμενη εργασία στην ουρά.
terminate_job: Τερματίζει μια εργασία με δεδομένο ID.
list_active_jobs: Εμφανίζει τις ενεργές εργασίες.
list_queued_jobs: Εμφανίζει τις εργασίες σε αναμονή.
